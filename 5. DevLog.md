## 📝 Prompt Improvement Summary (For Prompt Giver)

These are recurring issues that should be addressed in the master context prompt:

1. **Bootstrap removal** — Specify Tailwind CSS only from Task 1. Remove all Bootstrap references.
2. **`.env` file location** — Always at project ROOT (`/EquiSplit/.env`), not `config/.env`.
3. **`.env` loader** — Must be uncommented in `config/bootstrap.php` on project init.
4. **Table name** — Use `group_sessions` from the start (never `sessions`).
5. **OAuth pattern** — "Sign in with Google" is a GET link, never a form POST.
6. **Programmatic timestamps** — `joined_at` and `submitted_at` are set in `beforeSave()`, not DB defaults.
7. **Router import** — Always include `use Cake\Routing\Router;` in `Application.php`.
8. **`App.base` config** — Do NOT set manually; CakePHP auto-detects from folder name in Laragon.
9. **CakePHP ResultSet** — `array_map()` does not accept `Cake\ORM\ResultSet`; always convert with `foreach` or `->toArray()` first.
10. **Mermaid indentation** — Never indent mermaid code inside fenced blocks; the parser rejects leading whitespace on directives.
